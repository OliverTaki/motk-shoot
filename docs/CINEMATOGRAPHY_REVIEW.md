# Cinematography and review workflow

The **CINE** panel contains monitor-only analysis. Histogram, clipping zebra,
focus peaking, chroma key, and anamorphic desqueeze never alter browser captures,
tether originals, image-sequence exports, or movie frames. They are persisted so
the monitor returns to the same setup when the project opens.

Use a layer marked **Behind** as the chroma-key background. Image, video, shape,
text, and pen layers remain guide overlays and are never burned into footage.
Video reference time is `current exposure / fps`, adjusted by its frame offset.
Pen strokes are drawn directly in the viewport while their layer is selected.

The **REVIEW** panel sets playback speed and optional exposure IN/OUT points.
Loop repeats only that range. Hold **P** for a temporary live/selected-frame
pop-through. Source B can be another edit in the project or another tagged take
of the same production shot; choose Split A/B or full-frame A/B.

These are animation-monitor decisions. Editorial hand-off in Phase 6 will carry
the desqueeze note and conform information into dedicated text formats.
